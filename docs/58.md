# [58. 最后一个单词的长度](https://leetcode-cn.com/problems/length-of-last-word/)

### 思路

- 标签：`反转`、`循环`
- 反转字符串
- 如果长度 length 为 0，则代表最后字符为空，继续循环
- 时间复杂度：O(n)
- 空间复杂度：O(1)

### 代码

```python
## Python3
class Solution:
    def lengthOfLastWord(self, s: str) -> int:
        length = 0
        for x in range(len(s)):
            if s[len(s) - x - 1] != ' ':
                length += 1
            elif length == 0:
                continue
            else:
                break
        return length
```
