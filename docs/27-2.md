```
s1: [b, e, f, b]    m  
s2: [a, a, a, b, c, a, b, e, c, a, c, f, a, c]    n 
[a, a, a, c, a, c, a, c, a, c] 

目标： 从 s2 里面，移除所有 s1 里面出现的元素
返回： [a, a, a, c, a, c, a, c, a, c]
```

## 1、双重循环、额外数组

```Python
# Python3
class Solution:
    def removeAllElement(self, s1, s2):
        result = []
        y = 0
        for x in s2:   
            # 如果 x 不等于 s1 所有 的值，将其加入 result
            flag = true
            for i in range(s1):
                if x == s1[i]:
                    flag = false
                    break
            if flag:
                result[y] = x
                y = y + 1
           # 使用 while
```

```Python
# Python3
class Solution:
    def removeAllElement(self, s1, s2):
        result = []
        for x in s2:
            # 如果 x 不等于 s1 所有 的值，将其加入 result
            flag = True
            for y in s1:
                if x == y:
                    flag = False
                    break
            if flag:
                result.append(x)
        return result
```
- 时间复杂度：O(m * n)
- 空间复杂度：O(n)，基本上等于 O(n) 跟 m 没什么关系

## 2、单循环、额外数组

- map 的 get 方法时间复杂度为 O(1)，set 的判断也为 O(1)

```Python
# Python3
class Solution:
    def removeAllElement(self, s1, s2):
        result = []
        s1Set = {x for x in s1}
        for x in s2:
            if not x in s1Set:
                result.append(x)
        return result
```

- Java 的 HashSet 的 contains 时间复杂度为 O(1)

```Java
// Java
// Map map = new HashMap();
class Solution{
    public List<Character> removeAllElement(List<Character> s1, List<Character> s2) {
        Set<Character> s1Set = new HashSet<>();
//        for (Character x : s1) {
//            s1Set.add(x);
//        }
        s1Set.addAll(s1);
        ArrayList<Character> result = new ArrayList<>();
        for (Character y : s2) {
            if (!s1Set.contains(y)) {
                result.add(y);
            }
        }
        return result;
    }
}
```

- 时间复杂度：O(n+m)
- 空间复杂度：O(n)

## 3、一次循环，原地操作

```Python
# Python3
class Solution:
    def removeAllElement(self, s1, s2):
        s1Set = {x for x in s1}
        index = 0
        for x in s2:
            if not x in s1Set:
                s2[index] = x
                index = index + 1
        return s2[0:index]
```

```Java
// Java
class Solution {
    public List<Character> removeAllElement(List<Character> s1, List<Character> s2) {
        Set<Character> s1Set = new HashSet<>();
        s1Set.addAll(s1);
        int index = 0;
        for (Character y : s2) {
            if (!s1Set.contains(y)) {
                s2.set(index, y);
                index++;
            }
        }
        return s2.subList(0, index);
    }
}

```
- 时间复杂度：O(n)
- 空间复杂读：O(1)
