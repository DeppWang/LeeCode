查询当前部门工资最高的人

```sql
SELECT DepartmentId, Name, MAX(Salary) FROM Employee GROUP BY DepartmentId

{"headers": ["DepartmentId", "Name", "MAX(Salary)"], "values": [[1, "Joe", 90000], [2, "Henry", 80000]]}
```


**难点在于显示所有最大值**

1. 先找到所有的每个部门的最高薪资作为临时表

```SQL
SELECT DepartmentId, MAX(Salary) FROM Employee GROUP BY DepartmentId

{"headers": ["DepartmentId", "MAX(Salary)"], "values": [[1, 90000], [2, 80000]]}
```

2. 根据最高薪资查找相应姓名

```sql
SELECT e.DepartmentId, e.Name, e.Salary FROM (SELECT DepartmentId, MAX(Salary) AS Salary FROM Employee GROUP BY DepartmentId) tmp, Employee e WHERE tmp.DepartmentId = e.DepartmentId AND tmp.Salary = e.Salary ORDER BY e.DepartmentId

{"headers": ["DepartmentId", "Name", "Salary"], "values": [[1, "Jim", 90000], [1, "Max", 90000], [2, "Henry", 80000]]}
```

3. 转换部门 id 为 Name

```sql
SELECT d.Name AS Department, tmp2.Name AS Employee, tmp2.Salary FROM (SELECT e.DepartmentId, e.Name, e.Salary FROM (SELECT DepartmentId, MAX(Salary) AS Salary FROM Employee GROUP BY DepartmentId) tmp, Employee e WHERE tmp.DepartmentId = e.DepartmentId AND tmp.Salary = e.Salary) tmp2, Department d WHERE tmp2.DepartmentId = d.Id

{"headers": ["DepartmentId", "Name", "Salary"], "values": [[1, "Jim", 90000], [1, "Max", 90000], [2, "Henry", 80000]]}

{"headers": ["Department", "Employee", "Salary"], "values": [["IT", "Jim", 90000], ["Sales", "Henry", 80000], ["IT", "Max", 90000]]}
```
