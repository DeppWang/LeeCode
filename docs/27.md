# 27. 移除元素

https://leetcode-cn.com/problems/remove-element/

### 思路

- 标签：双指针
- 拷贝覆盖每一个值，跳过相同的值。

### 代码

```python
class Solution:
    def removeElement(self, nums: List[int], val: int) -> int:
        res = 0
        for num in nums:
            if (num != val):
                nums[res] = num
                res += 1
        return res
```
